One way to think about negative mass is to consider what would happen if you tried to push an object with negative mass. If you pushed an object with positive mass, it would accelerate in the direction you were pushing it. However, if you pushed an object with negative mass, it would accelerate in the opposite direction, as if you were pulling it instead of pushing it.
